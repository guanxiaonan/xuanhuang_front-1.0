### dmx-mm-iview 
是dmx-mm-platform 的前端部分，
使用了 基于Vue2的[iView Admin框架v1.2.2](https://iview.github.io/iview-admin)

### Install
```bush
// install dependencies
npm install
```
### Run
#### Development
```bush
npm run dev
```
#### Production(Build)
```bush
npm run build
```
